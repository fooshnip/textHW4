The Storyboard

The following graphic was created in D3 and data parsed from a raw movie script of Toy Story, found from The Internet Movie Script Database. It shows the actual flow of dialogue in the movie over time. The buttons below will trigger an animation that will play through the entire script! Mouseover the chart to see the character's quotes over time. ***Apologies for the current overflow on this one!!

The X-axis represents the index in which the character spoke. The circles are only sized for easier mouse-over functionality and size and overlap carry no particular importance (though overlap suggests a lot of activity in succession!).




The Neighborhood Map

Using the raw text data as seen above, we also clustered characters based on distance matrices constructed from term-document frequencies. The clusters were plotted using multi-dimensional scaling to create neighborhood maps.
